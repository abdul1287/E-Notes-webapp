# ENotes Web Application (JSP & Servlets)

ENotes is a **practice-based Java web application** built to understand the fundamentals of **JSP, Servlets, JDBC, and MVC architecture**.  
The application allows users to create, edit, delete, and manage personal notes in a simple web interface.

This project is developed for **learning and practice purposes** and is not intended for production use.
